Welcome to Blog Optimiser

In Claude Desktop, run: 

/blog-optimiser https://your-blog.com/blog

Your live progress, recommendations, and optimised articles will appear here automatically.

Prerequisites (connected in Claude Desktop's MCP settings)

  • • Peec AI MCP — for live gap data (peec.ai)
  • • Crawl4AI MCP — for page fetching and extraction
  • • Python 3.8+ — macOS bundles this; otherwise install from python.org

The plugin runs in "generic mode" if Peec isn't connected — recommendations are still produced using GEO best-practice rules and any competitor URLs you supply.
